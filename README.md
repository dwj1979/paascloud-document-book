## paascloud 实战项目

### 项目介绍
```
spring cloud + vue 全家桶，SPA动静分离，由用户认证中心、主数据中心、订单中心、对接平台组成.
```
### 作者介绍
```
Spring Cloud 爱好者,现就任于鲜易供应链平台研发部.
```
